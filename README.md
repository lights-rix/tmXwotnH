# 项目简介

## 核心功能
- 基于 YOLOv8 的蘑菇检测系统，支持识别 14 类蘑菇
- 包含完整源码、8300 张标注数据集、训练好的模型及训练结果
- 提供项目运行教程，配置成功环境可直接使用
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 支持自定义检测参数：置信度阈值、IOU阈值
- 提供可视化界面，实时显示目标位置、目标总数、置信度等信息
- 支持批量检测及结果保存
- 提供模型选择功能，支持显示标签和原图选项

## 适用场景
- 蘑菇种类识别研究
- 野外蘑菇鉴定辅助
- 农业蘑菇种植质量检测
- 生物学教学演示
- 生态调查与分类

## 技术特点
- 采用 Python 3.9 开发，依赖 opencv-python、PyQt5、ultralytics、torch1.9 等库
- 使用 Pycharm+Anaconda 或 VSCode+Anaconda 作为开发环境
- 基于 YOLOv8 目标检测算法，具有高精度和实时性
- 提供图形用户界面，操作简便直观
